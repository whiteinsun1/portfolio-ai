🧭 궤도 판정 원리
계획 곡선 = 시작 시드×복리 + 월 적립×적립복리 (설정한 연수익률 기준). 매달 기록한 실제 총자산이 이 곡선 위에 있으면 "앞섬 ✅", 아래면 "뒤처짐 ⚠️". 뒤처짐의 원인이 적립 부족인지 수익률 부족인지는 궤도 분석 AI가 분해해서 알려줍니다.

📓 매매 일지 → 보유 종목 자동 반영 규칙
· 매수: 이미 있는 종목이면 수량 증가 + 평단 가중평균 재계산 / 없으면 자동 등록
· 매도: 수량 차감, 0이 되면 제거 여부 물어봄
· 한국 주식: 🔍 버튼으로 한글 이름 검색 → 코드 자동 입력 / 미국 주식: 티커 직접 입력

🏛️ 코어/새틀라이트 판정
기본은 티커 자동 판정(QQQ·SPY·VOO·IVV = 코어). 아니라면 배분 카드에서 "전환" 버튼으로 직접 지정하세요. 한 번 지정하면 구글시트에 저장됩니다.

💾 데이터 저장 위치
모든 데이터는 내 구글시트(Stocks·Snapshots·Journal·Settings 탭)에 저장되고, 이 브라우저에도 캐시됩니다. 다른 기기에서 열어도 시트에서 자동으로 불러옵니다.

🔗 V1과의 관계
V1(index.html)은 종목별 AI 심층 분석·기술지표 진단 전용, V2는 목표 관리·일지·자산 추적 전용입니다. 같은 구글시트를 공유하므로 어느 쪽에서 수정해도 서로 반영됩니다.

💎 프리미엄 리포트
평소 분석은 전부 무료 모델이고, 프리미엄 버튼만 Claude Opus 유료 호출(건당 약 150~200원)입니다. Settings 탭에 anthropic_key를 추가해야 작동하며, 누를 때만 과금됩니다. 월급날 배분·매수/매도 갈림길 같은 실제 의사결정 직전에 쓰는 용도예요.
