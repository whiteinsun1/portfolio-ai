# 📈 AI 주식 포트폴리오 관리기 (V1 진단기 + V2 대시보드)

평범한 직장인이 **10년 장기 복리로 시드머니를 우상향**시켜 경제적 자유에 도달하는 것을 돕는 개인용 웹 앱입니다.
서버 없이 GitHub Pages(무료)에서 돌아가고, 데이터는 전부 **내 구글시트**에 저장됩니다.

| 페이지 | 주소 | 역할 |
|--------|------|------|
| **V1 진단기** | [`index.html`](https://whiteinsun1.github.io/portfolio-ai/) | 종목별 AI 심층 분석 · 기술지표 진단 · 분석 히스토리 |
| **V2 대시보드** | [`v2.html`](https://whiteinsun1.github.io/portfolio-ai/v2.html) | 10년 목표 궤도 추적 · 월간 자산 스냅샷 · 매매 일지 · 행동 복기 |

두 페이지는 **같은 구글시트를 공유**하므로 어느 쪽에서 수정해도 서로 반영됩니다.

## 🏗️ 전체 구조

```
[브라우저: index.html / v2.html]  ← GitHub Pages (정적 호스팅, 무료)
        │
        ├── 시세 조회: 네이버 금융(국내, 무료) + Twelve Data(미국·환율, 무료 키)
        ├── AI 분석: OpenRouter 무료 모델 4종 폴백 호출
        │
        └── [Google Apps Script 웹앱]  ← 서버 역할 (무료)
                  │
                  ├── 데이터 저장/불러오기 (구글시트)
                  ├── 화~토 오전 7시대: 자동 종합분석
                  ├── 일요일 오전 9시대: 텔레그램 주간 리포트
                  └── 매월 1일 오전 10시대: 스냅샷 기록 리마인드 (선택)
```

- **비용**: 전부 무료 티어로 동작. (선택: OpenRouter 유료 백업 모델 추가 가능)
- **보안**: API 키는 소스 코드에 없음. 내 구글시트 Settings 탭 + 브라우저 localStorage에만 존재.

## 🚀 처음 설정 (한 번만)

1. **구글시트 만들기** — 빈 시트 하나 생성, URL에서 시트 ID 복사
2. **Apps Script 배포** — [script.google.com](https://script.google.com) → 새 프로젝트 → `Code.gs` 내용 붙여넣기
   - 3번째 줄 `SECRET_KEY`를 나만의 비밀키로 변경
   - 6번째 줄 `SHEET_ID`를 내 시트 ID로 변경
   - 배포 → 새 배포 → 웹 앱 → "나만" 실행, "모든 사용자" 액세스 → 배포 → **웹앱 URL 복사**
3. **API 키 발급 (무료)**
   - [OpenRouter](https://openrouter.ai) → API 키 (AI 분석용)
   - [Twelve Data](https://twelvedata.com) → API 키 (미국 시세·환율용, 카드 불필요)
   - 발급한 키를 구글시트 **Settings 탭**에 입력: `openrouter_key`, `twelvedata_key`
4. **앱 접속** — V1 설정 패널에 Apps Script URL + 비밀키 입력 → "구글시트에서 불러오기"
5. **자동화 트리거 설치** — Apps Script 편집기에서 함수 선택 후 실행(▶):
   - `setupDailyTrigger` → 화~토 아침 자동 종합분석
   - `setupWeeklyTelegramTrigger` → 일요일 텔레그램 리포트 (봇 토큰 필요, 아래 참고)
   - `setupMonthlySnapshotReminder` → 매월 1일 스냅샷 리마인드 (선택)

### 텔레그램 연동 (선택)
1. 텔레그램 @BotFather → `/newbot` → 토큰 발급
2. 만든 봇에게 아무 메시지 전송 → `https://api.telegram.org/bot<토큰>/getUpdates` 열어 `"chat":{"id":숫자}` 확인
3. Settings 탭에 `telegram_bot_token`, `telegram_chat_id` 추가

## 📊 V1 진단기 — 기능과 로직

### 기능
- 종목 등록(티커·수량·평단) / 수정 / 삭제
- **종목별 AI 분석 2종**: 🔍 종목 심층 진단 / 📈 10년 복리 관점 분석
- **전체 종합분석**: 코어-새틀라이트 구조 진단 + 매수형 리밸런싱 제안
- 분석 히스토리 누적 + 수익률 꺾은선 차트 + CSV 내보내기

### 분석 프롬프트에 들어가는 것
1. **투자자 프로필**: 경제적 자유 목표, 코어-새틀라이트 전략, 월 100~150만원 적립
2. **운용 원칙 5개조** (모든 AI 제안이 이 원칙을 벗어나지 못하게 강제):
   - 복리의 수학: -50% 손실은 +100% 수익으로만 복구 → 큰 손실 회피 우선
   - **물타기 금지, 불타기 원칙**: 하락 추세 종목 평단 낮추기 금지
   - 기회비용: 새틀라이트는 QQQM 초과수익 기대가 있을 때만 보유
   - 집중 한도: 새틀라이트 단일 종목 15% 이하
   - 월 투입 우선순위: 코어 유지 → 검증된 새틀라이트 → 현금 대기도 유효
3. **기술지표** (미국 주식, 자동 조회): 52주 밴드 위치, 20/50/200일 이동평균선, RSI(14)
   → 손절선을 "현재가 -8%" 같은 임의 숫자가 아니라 **실제 지지선(200일선·52주 저점) 기준**으로 제안하게 함

### AI 모델
OpenRouter 무료 모델 4종을 순서대로 시도 (혼잡 시 자동 전환):
`llama-3.3-70b` → `gpt-oss-120b` → `qwen3-next-80b` → `nemotron-3-super`

## 🎯 V2 대시보드 — 기능과 로직

### 핵심 철학
장기 투자 성패의 실제 순위는 ① 저축률·적립 지속성 ② 행동 갭 교정 ③ 자산배분 유지 ④ 종목 선택 순.
V1이 ④를 담당하고, **V2는 ①②③을 담당**합니다.

### 카드별 기능

**🧭 목표까지의 궤도**
- 목표 금액·기간·예상수익률·월 적립액 설정 → **계획 곡선 vs 실제 자산 곡선** 차트
- 역산 엔진: 현재 설정으로 10년 뒤 예상액 / 목표 도달에 필요한 월 적립액 / 필요 연수익률 자동 계산
- 계획 곡선 수식: `계획(n개월) = 시드 × (1+i)ⁿ + 월적립 × ((1+i)ⁿ − 1)/i`, `i = (1+연수익률)^(1/12) − 1`

**📅 이번 달 체크**
- 🔄 자동 계산: 등록 종목 시세 조회 + USD/KRW 환율 자동 적용 → 주식 자산 원화 합산
- 기타 자산(현금·예금) 입력 → 월 1회 스냅샷 저장 → 궤도 차트에 점이 찍힘
- 적립 실행 여부도 함께 기록

**⚖️ 코어-새틀라이트 배분**
- 코어 비중 게이지 (50~70% 밴드 마커) — 이탈 시 경고
- 새틀라이트 단일 종목 15% 초과 경고
- 종목별 "전환" 버튼으로 코어/새틀라이트 역할 직접 지정 (기본: QQQ·SPY·VOO·IVV = 코어 자동 판정)

**📓 매매 일지**
- 매매마다 이유·손절선·목표가·**그때의 감정**(확신/불안/FOMO/공포) 기록
- **보유 종목 자동 반영**: 매수 → 수량 증가 + 평단 가중평균 재계산 (신규면 자동 등록) / 매도 → 수량 차감
- 한국 주식은 🔍 버튼으로 **한글 이름 검색** → 종목코드 자동 입력 (네이버 자동완성 API)

**🤖 AI 분석 V2**
- 🛰️ **궤도 종합분석**: 계획 대비 차이의 원인을 분해(적립 부족 vs 수익률 부족) + 향후 12개월 우선순위 + 이번 달 행동 3줄
- 🪞 **행동 복기**: 매매 일지에서 물타기·FOMO·손절 미준수 패턴을 찾아 교정 처방 (개인투자자의 행동 갭 연 1~2%p를 줄이는 목적)

## 🔧 문제 해결

| 증상 | 원인 | 해결 |
|------|------|------|
| "모든 무료 모델이 응답하지 않습니다" | OpenRouter 무료 모델 혼잡(429) | 몇 분 뒤 재시도. 잦으면 유료 백업 모델 추가 검토 |
| 401 Missing Authentication | API 키가 이 브라우저에 없음 | 설정에서 Apps Script URL·비밀키 입력 → 불러오기 |
| 키를 불러왔는데도 401 | Settings 탭 키 값에 URL 등 잡문자 포함 | 시트에서 키 값만 남기고 정리 후 재동기화 |
| 시세 자동 계산 실패 | Twelve Data 무료 한도(분당 8회) 초과 | 1분 뒤 재시도 |
| 앱 수정했는데 반영 안 됨 | Apps Script 재배포 누락 | 배포 → 배포 관리 → ✏️ → 새 버전 → 배포 |
| 텔레그램 안 옴 | 토큰/chat_id 오류 | Settings 탭 `telegram_bot_token`·`telegram_chat_id` 확인 |
