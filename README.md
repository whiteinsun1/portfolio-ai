# 📈 AI 주식 포트폴리오 관리기 (V1 진단기 + V2 대시보드)

평범한 직장인이 **10년 장기 복리로 시드머니를 우상향**시켜 경제적 자유에 도달하는 것을 돕는 개인용 웹 앱입니다.
서버 없이 GitHub Pages(무료)에서 돌아가고, 데이터는 전부 **내 구글시트**에 저장됩니다.

| 페이지 | 주소 | 역할 |
|--------|------|------|
| **V1 진단기** | [`index.html`](https://whiteinsun1.github.io/portfolio-ai/) | 종목별 AI 심층 분석 · 기술지표 진단 · 분석 히스토리 |
| **V2 대시보드** | [`v2.html`](https://whiteinsun1.github.io/portfolio-ai/v2.html) | 10년 목표 궤도 추적 · 월간 자산 스냅샷 · 매매 일지 · 행동 복기 |

두 페이지는 **같은 구글시트를 공유**하며 상단 링크로 서로 오갈 수 있습니다. 어느 쪽에서 수정해도 서로 반영됩니다.
PC와 **스마트폰 모두 대응**합니다 — 폰에서는 카드가 한 열로 쌓이고, 리포트 안의 표는 표만 가로로 밀어서 볼 수 있습니다.

## 🏗️ 전체 구조

```
[브라우저: index.html / v2.html]  ← GitHub Pages (정적 호스팅, 무료)
        │
        ├── 시세 조회: 네이버 금융(국내, 무료) + Twelve Data(미국·환율, 무료 키)
        ├── 종목 검색: 네이버 자동완성 (한글 이름 → 종목코드)
        ├── AI 분석: OpenRouter 무료 모델 4종 폴백 호출
        │
        └── [Google Apps Script 웹앱]  ← 서버 역할 (무료)
                  │
                  ├── 데이터 저장/불러오기 (구글시트, 동시 저장 잠금 처리)
                  ├── 화~토 오전 7시대: 자동 종합분석
                  ├── 일요일 오전 9시대: 텔레그램 주간 리포트 (+목표 진척)
                  └── 매월 1일 오전 10시대: 스냅샷 기록 리마인드
```

- **비용**: 전부 무료 티어로 동작
- **보안**: API 키는 소스 코드에 없음. 내 구글시트 Settings 탭 + 브라우저 localStorage에만 존재

## 🚀 처음 설정 (한 번만)

1. **구글시트 만들기** — 빈 시트 하나 생성, URL에서 시트 ID 복사
2. **Apps Script 배포** — [script.google.com](https://script.google.com) → 새 프로젝트 → `Code.gs` 내용 붙여넣기
   - 3번째 줄 `SECRET_KEY`를 나만의 비밀키로 변경
   - 6번째 줄 `SHEET_ID`를 내 시트 ID로 변경
   - 배포 → 새 배포 → 웹 앱 → "나만" 실행, "모든 사용자" 액세스 → 배포 → **웹앱 URL 복사**
3. **API 키 발급 (무료)**
   - [OpenRouter](https://openrouter.ai) → API 키 (AI 분석용)
   - [Twelve Data](https://twelvedata.com) → API 키 (미국 시세·환율용, 카드 불필요)
   - 발급한 키를 구글시트 **Settings 탭**에 입력: `openrouter_key`, `twelvedata_key`
   - (선택) [Anthropic Console](https://console.anthropic.com) → API 키 + 크레딧 충전 → Settings 탭에 `anthropic_key`
     — V2의 💎 프리미엄 의사결정 리포트용. 없어도 나머지 기능은 전부 동작
4. **앱 접속** — V1 설정 패널에 Apps Script URL + 비밀키 입력 → "구글시트에서 불러오기"
5. **자동화 트리거 설치** — Apps Script 편집기에서 함수 선택 후 실행(▶):
   - `setupDailyTrigger` → 화~토 아침 자동 종합분석 (트리거 5개 생성)
   - `setupWeeklyTelegramTrigger` → 일요일 텔레그램 리포트
   - `setupMonthlySnapshotReminder` → 매월 1일 스냅샷 리마인드
   - 설치 확인: 왼쪽 ⏰ 트리거 메뉴에 총 **7개** 보이면 정상
6. **데이터 첫 입력** — 아래 [🏁 첫 입력 가이드](#-첫-입력-가이드-설치-끝난-직후-약-15분) 순서대로 진행

> 💡 트리거는 한 번 설치하면 계속 유지됩니다. 코드를 수정·재배포해도 다시 실행할 필요 없음.
> 단, `setup...` 함수 **내용 자체**가 바뀐 업데이트를 받았을 때만 해당 함수를 다시 실행하세요.

### 텔레그램 연동 (선택)
1. 텔레그램 @BotFather → `/newbot` → 토큰 발급
2. 만든 봇에게 아무 메시지 전송 → `https://api.telegram.org/bot<토큰>/getUpdates` 열어 `"chat":{"id":숫자}` 확인
3. Settings 탭에 `telegram_bot_token`, `telegram_chat_id` 추가

## 🏁 첫 입력 가이드 (설치 끝난 직후, 약 15분)

설치가 끝났다면 아래 순서로 데이터를 처음 채웁니다. **이 순서대로 해야 각 단계가 다음 단계의 재료가 됩니다.**

### ① 보유 종목 등록 — V1에서
- V1 접속 → 종목 등록 폼에 티커·수량·평균매입가 입력
  - 미국 주식: 티커 그대로 (예: `QQQM`)
  - 한국 주식: 6자리 종목코드 (예: 삼성전자 `005930`) — 모르면 네이버 검색
- 보유 종목을 **전부** 등록해야 배분·스냅샷이 정확해집니다

### ② 목표 설정 — V2에서
- V2 접속 → 🧭 카드의 "⚙️ 목표 설정" 펼치기
- **시작 시드**(현재 총자산)를 꼭 입력 → 목표 금액·목표일·예상 연수익률·월 적립액 입력 → 저장
- 저장하면 역산 결과("현재 설정이면 10년 뒤 얼마")가 바로 나옵니다.
  **여기서 목표가 비현실적으로 나오면 숫자를 조정하세요** — 현실적인 계획을 마주하는 것이 첫걸음입니다

### ③ 역할 지정 — V2 배분 카드에서
- 지수 ETF인데 새틀라이트로 표시된 종목이 있으면 "코어로" 버튼으로 교정
- (KODEX 미국나스닥100 같은 국내 상장 지수 ETF는 자동 판정이 안 되므로 직접 지정 필요)

### ④ 첫 스냅샷 — V2 이번 달 체크에서
- "🔄 자동 계산" 클릭 → 기타 자산(현금·예금) 입력 → 적립액 입력 → 저장
- 이 순간 10년 그래프에 **첫 번째 점**이 찍힙니다 🎯

### ⑤ 동작 확인
- 🛰️ 궤도 종합분석 한 번 실행 → 결과가 나오고 "🗂️ 지난 분석 기록"에 1건 쌓이면 전부 정상

### 루틴의 핵심 원칙
- **스냅샷은 거르지 않기** — 한 달에 점 하나. 이 점들이 모여야 "계획 vs 실제" 판정이 의미를 가집니다. 월초에 텔레그램 리마인드가 옵니다.
- **일지는 매수 전에 쓰는 게 최고** — "왜 사는지"를 글로 쓰다 보면 FOMO 매수가 걸러집니다. 최소한 매수 직후라도 꼭.
- **일요일 리포트의 "이번 주 할 일"은 참고 자료** — AI 제안은 조언이지 명령이 아닙니다. 최종 판단은 항상 본인이.
- **행동 복기는 일지 3건부터 가능** — 기록이 쌓일수록 정확해집니다. 분기 1회면 충분.

## 📊 V1 진단기 — 기능과 로직

### 기능
- 종목 등록(티커·수량·평단) / 수정 / 삭제
- **종목별 AI 분석 2종**: 🔍 종목 심층 진단 / 📈 10년 복리 관점 분석
- **전체 종합분석**: 코어-새틀라이트 구조 진단 + 매수형 리밸런싱 제안
- 분석 히스토리 누적 + 수익률 꺾은선 차트 + CSV 내보내기

### 분석 프롬프트에 들어가는 것
1. **투자자 프로필**: 경제적 자유 목표, 코어-새틀라이트 전략, 월 100~150만원 적립
2. **운용 원칙 5개조** (모든 AI 제안이 이 원칙을 벗어나지 못하게 강제):
   - 복리의 수학: -50% 손실은 +100% 수익으로만 복구 → 큰 손실 회피 우선
   - **물타기 금지, 불타기 원칙**: 하락 추세 종목 평단 낮추기 금지
   - 기회비용: 새틀라이트는 QQQM 초과수익 기대가 있을 때만 보유
   - 집중 한도: 새틀라이트 단일 종목 15% 이하
   - 월 투입 우선순위: 코어 유지 → 검증된 새틀라이트 → 현금 대기도 유효
3. **기술지표** (미국 주식, 자동 조회): 52주 밴드 위치, 20/50/200일 이동평균선, RSI(14)
   → 손절선을 임의 %가 아니라 **실제 지지선(200일선·52주 저점) 기준**으로 제안하게 함
4. **코어/새틀라이트 역할 태그**: V2에서 지정한 역할이 종합분석 프롬프트에 [코어]/[새틀라이트]로 표시됨
5. **시장 국면**: 나스닥100(QQQ)이 200일선 위/아래인지 자동 판정 (12시간 캐시)
   → 상승 국면에만 새틀라이트 불타기 허용, 조정 국면엔 코어 적립·현금 대기 우선
6. **직전 분석 결론 (연속성)**: 같은 대상의 마지막 분석 발췌를 함께 제공
   → "판단이 바뀌었다면 실제로 무엇이 변했는지 명시하라. 변한 게 없으면 뒤집지 마라"
7. **대기 현금** (종합분석): V2 스냅샷의 기타자산을 읽어 집행 가능한 금액 범위 안에서만 제안하게 함

### 편향 방지 장치 (모든 분석 공통)
- **뒤집기 질문(플립 테스트)**: "오늘 이 종목이 없다면 지금 새로 사겠는가?" — 관성 보유·본전 심리 차단
- **반대 논거 우선**: bear case를 먼저 세우고 데이터로 반박 가능한지 검증 (좋은 게 좋다는 답변 방지)
- **📍 다음 검증 체크포인트**: 모든 분석이 "___가 ___이면 이번 판단은 틀린 것"이라는 반증 조건으로 끝남
  → 다음 분석에서 직전 판단이 자동으로 검증되는 자기교정 루프

### AI 모델
OpenRouter 무료 모델 4종을 순서대로 시도 (혼잡 시 자동 전환):
`llama-3.3-70b` → `gpt-oss-120b` → `qwen3-next-80b` → `nemotron-3-super`

## 🎯 V2 대시보드 — 기능과 로직

### 핵심 철학
장기 투자 성패의 실제 순위는 ① 저축률·적립 지속성 ② 행동 갭 교정 ③ 자산배분 유지 ④ 종목 선택 순.
V1이 ④를 담당하고, **V2는 ①②③을 담당**합니다.

### 카드별 기능

**🧭 목표까지의 궤도**
- 목표 금액·기간·예상수익률·월 적립액 설정 (금액 입력 시 천 단위 콤마 자동 표시)
- **계획 곡선 vs 실제 자산 곡선** 차트 + 궤도 판정(앞섬 ✅/뒤처짐 ⚠️)
- 역산 엔진: 현재 설정으로 10년 뒤 예상액 / 목표 도달에 필요한 월 적립액 / 필요 연수익률
- 계획 곡선 수식: `계획(n개월) = 시드 × (1+i)ⁿ + 월적립 × ((1+i)ⁿ − 1)/i`, `i = (1+연수익률)^(1/12) − 1`

**📅 이번 달 체크**
- 🔄 자동 계산: 전 종목 실시간 시세 + USD/KRW 환율 자동 적용 → 주식 자산 원화 합산
- 기타 자산(현금·예금) 입력 → 월 1회 스냅샷 저장 → 궤도 차트에 점이 찍힘

**⚖️ 코어-새틀라이트 배분**
- 코어 비중 게이지 (50~70% 밴드 마커), 이탈 시 경고 / 새틀라이트 단일 15% 초과 경고
- **코어 그룹 위·새틀라이트 그룹 아래**, 각 그룹 안에서 비중 큰 순 정렬, 그룹 합계 표시
- 종목별 "코어로/새틀로" 버튼으로 역할 직접 지정 (기본: QQQ·SPY·VOO·IVV 자동 판정)
- **🔄 시세 새로고침**: 실시간 시세로 보유금액·비중 갱신 + "시세 기준: n분 전" 표시
  (가격 우선순위: 실시간 캐시 → 마지막 분석 가격 → 매입가)

**📓 매매 일지**
- 매매마다 이유·손절선·목표가·**그때의 감정**(확신/불안/FOMO/공포) 기록
- **보유 종목 자동 반영**: 매수 → 수량 증가 + 평단 가중평균 재계산 (신규면 자동 등록) / 매도 → 수량 차감, 0이면 제거 확인
- 한국 주식: 🔍 버튼으로 **한글 이름 검색** → 종목코드 자동 입력 / 미국 주식: 티커 직접 입력

**🤖 AI 분석 V2**
- 🛰️ **궤도 종합분석**: 계획 대비 차이의 원인 분해(적립 부족 vs 수익률 부족) + 향후 12개월 우선순위 + 이번 달 행동 3줄 + 검증 체크포인트
  — 직전 궤도 분석·시장 국면·대기 현금을 함께 제공해 일관성 있고 집행 가능한 처방이 나오게 함
- 🪞 **행동 복기**: 매매 일지에서 물타기·FOMO·손절 미준수 패턴을 찾아 교정 처방 (일지 3건 이상부터)
  — 두 번째 복기부터는 **"지난 복기 이행 점검"** 섹션이 맨 앞에 붙어, 지난번 지적한 습관이 고쳐졌는지 먼저 평가 (행동 갭 교정의 핵심 루프)
- 💎 **프리미엄 의사결정 리포트 (유료, 선택)**: 버튼을 누를 때만 Claude Opus(`claude-opus-4-8`)를 직접 호출 (건당 약 150~200원)
  — 실시간 시세·목표 궤도·대기 현금·매매 일지·직전 리포트·시장 국면을 **전부** 넣고, 시나리오 분석(강세/기준/약세 각 확률·대응),
  종목별 판정표, 조건부 트리거가 붙은 실행 계획, **자기 결론 반박**까지 요구하는 의사결정 보고서
  — 평소 루틴은 전부 무료 모델, 이 버튼만 유료. Settings 탭에 `anthropic_key` 필요. 월급날 배분·매수/매도 갈림길용
- **🗂️ 지난 분석 기록**: 실행할 때마다 자동 저장(최대 30건), 접이식 리스트로 과거 분석 조회·삭제 가능

**❓ 도움말** — 매달 루틴, 계산 원리, 데이터 저장 위치를 앱 안에서 바로 확인

## 🔧 문제 해결

| 증상 | 원인 | 해결 |
|------|------|------|
| "모든 무료 모델이 응답하지 않습니다" | OpenRouter 무료 모델 혼잡(429) | 몇 분 뒤 재시도. 잦으면 유료 백업 모델 추가 검토 |
| 401 Missing Authentication | API 키가 이 브라우저에 없음 | 설정에서 Apps Script URL·비밀키 입력 → 불러오기 |
| 키를 불러왔는데도 401 | Settings 탭 키 값에 URL 등 잡문자 포함 | 시트에서 키 값만 남기고 정리 후 재동기화 |
| 시세 자동 계산/새로고침 실패 | Twelve Data 무료 한도(분당 8회) 초과 | 1분 뒤 재시도 |
| 한글 종목 검색 실패 | 네이버 검색 프록시 일시 오류 | 6자리 종목코드 직접 입력 |
| 앱 수정했는데 반영 안 됨 | Apps Script 재배포 누락 | 배포 → 배포 관리 → ✏️ → 새 버전 → 배포 |
| 시트에 종목이 중복으로 보임 | 과거 버전의 동시 저장 사고 흔적 | 최신 Code.gs 적용 후 앱에서 아무 저장이나 한 번 실행하면 자동 정리 |
| 텔레그램 안 옴 | 토큰/chat_id 오류 | Settings 탭 `telegram_bot_token`·`telegram_chat_id` 확인 |
| 프리미엄 버튼이 키 없다고 함 | `anthropic_key` 미등록 | Settings 탭에 `anthropic_key` 추가 후 새로고침 (크레딧 충전도 필요) |
| 프리미엄 호출 401/400 | 키 오류 또는 크레딧 소진 | console.anthropic.com에서 키·크레딧 확인 |
